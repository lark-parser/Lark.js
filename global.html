    inplace_transformer()

    Propagate ambiguous intermediate nodes and their derivations up to the current rule.

    In general, converts

rule
  _iambig
    _inter
      someChildren1
      ...
    _inter
      someChildren2
      ...
  someChildren3
  ...

to

_ambig
  rule
    someChildren1
    ...
    someChildren3
    ...
  rule
    someChildren2
    ...
    someChildren3
    ...
  rule
    childrenFromNestedIambigs
    ...
    someChildren3
    ...
  ...

propagating up any nested '_iambig' nodes along the way.
